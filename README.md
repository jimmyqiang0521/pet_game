# 电子宠物游戏

一个面向小朋友的在线电子宠物养成游戏。

## 功能特点

- 🐱 选择领养小猫、小狗、小兔
- 🍖 喂食、玩耍、洗澡、睡觉
- 📊 属性系统：饱食度、心情值、清洁度、体力值
- 🏪 商店系统：购买食物、玩具、装饰
- 🏆 成就系统：解锁各种成就徽章
- 💾 本地存储：自动保存游戏进度

## 技术栈

- HTML5 + CSS3 + JavaScript
- 纯前端实现，无需后端
- localStorage 数据持久化
- 响应式设计，移动端优先

## 如何使用

1. 打开网站
2. 选择一只宠物领养
3. 定期照顾你的宠物
4. 赚取金币购买道具
5. 解锁成就徽章

## 本地运行

直接在浏览器中打开 `index.html` 文件即可。

或者使用本地服务器：
```bash
# Python
python -m http.server 8080

# Node.js
npx http-server
```
